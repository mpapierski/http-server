## Overview

`http-server` is a multi-platform HTTP server library with a focus on asynchronous I/O.

## Features

* External event loop support
* HTTP/1.1 support
* C89 compatible

## Build Instructions

	$ mkdir build
	$ cd build
	$ cmake ..
	$ make

Building tests:

	$ make tests

Coverage report:

	$ make coverage

(requires lcov)
